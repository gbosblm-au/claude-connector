# data/

Place your LinkedIn export files in this directory:

- **connections.csv** - Your LinkedIn connections export (see README for instructions)
- **profile.json** - Your LinkedIn profile data (optional, see example below)

## profile.json example

```json
{
  "name": "Your Full Name",
  "headline": "Senior Engineer at Acme Corp",
  "location": "Melbourne, VIC, Australia",
  "industry": "Technology",
  "summary": "Experienced engineer with a background in cloud infrastructure and distributed systems.",
  "email": "you@example.com",
  "linkedinUrl": "https://www.linkedin.com/in/yourprofile",
  "currentCompany": "Acme Corp",
  "currentPosition": "Senior Engineer"
}
```
